# Notebooks

This directory contains Jupyter notebooks used for exploratory data analysis (EDA), event impact modeling, and forecasting.

## Notebook List
- **Task 1: Data Exploration and Enrichment**: Initial data loading, structure check, and enrichment.
- **Task 2: Exploratory Data Analysis**: Deep dive into access and usage trends, correlation analysis, and data quality check.
- **Task 3: Event Impact Modeling**: Modeling the impact of policy and product launches using comparable country evidence.
- **Task 4: Forecasting Access and Usage**: Forecasting time series trends for the years 2025-2027.
